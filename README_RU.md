## Возможности

### ✍️ Markdown-редактор на CodeMirror 6
- **Подсветка синтаксиса** CodeMirror 6
- **Панель форматирования** — для тех, кто не помнит все сочетания клавиш
- **Вики-ссылки** `[[Имя Персонажа]]` — клик открывает Wiki-запись
- **Режим пишущей машинки** — курсор всегда по центру экрана
- **Режим фокуса** — всё, кроме активного абзаца, приглушается (как в iA Writer)
- **Zen-режим** — скрывает весь интерфейс, только текст
- **Ширина колонки** — Default, Narrow (60ch), Medium (72ch), Wide (90ch)
- **Нумерация строк** — вкл/выкл в настройках
- **Предпросмотр** — живой рендеринг markdown с поддержкой KaTeX-математики
- **Автосохранение** — дебаунс-сохранение после каждого нажатия клавиши
- **Эффект faded маркеров** — `**`, `*`, `#`, `[]` исчезают как в iA Writer

### 🗂️ Система контекста (Персонажи, Места, Даты, Предметы)

| Тип | Описание |
|------|----------|
| `character` | Персонажи — живые, мёртвые или обречённые на трагедию |
| `place` | Локации мира — куда герои доберутся... в теории |
| `date` | Временны́е якоря вашего мира |
| `item` | Артефакты, оружие, макгаффины — всё, что движет сюжетом |

Каждая запись содержит **details**, **relations**, **group** и **notes**. Для каждого типа есть шаблон.

### 🕸️ Карта отношений персонажей
- **SVG-холст** с панорамированием, зумом и перетаскиванием
- **Кривые Безье** — плавные линии между персонажами (не прямые)
- **Группы** — персонажи объединяются с полупрозрачным фоном
- **Режим связывания** — клик на двух персонажей создаёт отношение
- **Space + перетаскивание** для панорамирования (как в Figma)
- **Ctrl+0** — вписать все узлы в экран
- **Двойной клик** по узлу — редактировать персонажа

Типы отношений и их цвета:
| Тип | Цвет |
|------|-------|
| Союзник | Зелёный |
| Враг | Красный |
| Семья | Синий |
| Нейтрально | Серый |
| Романтика | Жёлтый |
| Соперник | Фиолетовый |

### 📚 Wiki-просмотр
- Детальный просмотр персонажа, места, предмета или события
- Редактируемые свойства, заметки и отношения
- Цветные бейджи отношений
- Быстрый переход к Карте отношений

### ⏱️ Хронология
- Хронологические события с датами, цветами, метками и заметками
- Привязка персонажей, мест и предметов к событиям
- Дата окончания и примечания к датам

### 🌍 Построение мира
- Структурированные записи с заголовком, содержанием и категорией
- Сетка для просмотра
- Свободные категории (География, Магия, История и т.д.)

### 📋 Канбан-доска
- Колонки и карточки для планирования сюжета
- Цветные метки с редактируемой легендой
- Встроенные формы добавления/редактирования

### 🔍 Полнотекстовый поиск
- `Ctrl+Shift+F` — поиск по всем главам и контексту
- Результаты сгруппированы по главам с количеством совпадений

### 📸 История версий (снапшоты)
- Ручные снапшоты с метками
- Авто-снапшоты с настраиваемым интервалом
- Счётчик слов, символов и глав для каждого снапшота
- Восстановление любой версии в один клик

### 🐻 Импорт/Экспорт Bear
- Импорт и экспорт в формате Bear app (ZIP-архивы)
- Бесшовная миграция между платформами

### 🌐 Интернационализация — 3 языка
- **English**, **Español**, **Русский**
- **273 типобезопасных ключа перевода** — авто-генерация из `en.json`
- Все строки интерфейса через `t('key')` — никакого хардкода

### 🎨 3 темы Fluent Acrylic
- **Light** — чистый синий акцент, высокий контраст
- **Dark** — спокойные тёмные поверхности, голубой акцент
- **Paper** — приглушённый тёплый серый, оливковый акцент, для комфорта глаз (низкий контраст, нет чистого белого и чёрного)

### 📤 Экспорт
- **DOCX** — библиотека `docx`
- **PDF** — генерируется на Rust (`printpdf` + `font-kit` для кириллических шрифтов)
- **.bear** архивы для бекапа и переносимости

### 📊 Строка состояния
- Таймер сессии (сбрасывается при смене главы)
- Счётчик слов в реальном времени
- Прогресс-бар дневной цели (по умолчанию: 500 слов)
- Счётчик streak'а — дней подряд с писательской активностью
- Индикатор «Сохранено X назад» с автообновлением

## Технологический стек

| Слой | Технология |
|-------|-----------|
| **Фронтенд** | React 19, TypeScript (`strict: true`), Vite 8 |
| **Редактор** | CodeMirror 6 + 5 кастомных ViewPlugin'ов |
| **Состояние** | Zustand (3 хранилища) |
| **Стили** | Tailwind CSS 4 + компонентные CSS-файлы (Fluent Acrylic) |
| **Иконки** | Lucide React |
| **Бэкенд** | Rust (Tauri 2.x) |
| **Математика** | KaTeX |
| **Экспорт** | `docx` (DOCX), `printpdf` (PDF) |
| **Тестирование** | Vitest (16 юнит-тестов), Playwright (e2e smoke) |

## Формат файлов

Книги хранятся как **папки** с markdown-файлами глав и двумя JSON-конфигами:

### `.book.json` — Метаданные
```json
{
  "title": "Мой роман",
  "author": "Имя автора",
  "genre": "Фэнтези",
  "bookType": "Novel",
  "description": "История о...",
  "createdAt": "2026-01-01T00:00:00.000Z",
  "chapters": [
    { "id": "uuid", "name": "Глава 1", "file": "Глава 1.md" }
  ]
}
```

### `.context.json` — Персонажи, хронология, заметки, мир, канбан
```json
{
  "context": [
    {
      "name": "Герой",
      "type": "character",
      "details": { "Age": "30", "Gender": "Мужской" },
      "relations": [{ "name": "Союзник", "type": "ally" }],
      "group": "Протагонисты",
      "notes": "",
      "_x": 100, "_y": 200
    }
  ],
  "timelineData": [],
  "notes": [],
  "worldData": [],
  "kanbanData": { "columns": [] }
}
```

## Горячие клавиши

| Сочетание | Действие |
|----------|--------|
| `Ctrl+K` | Командная палитра |
| `Ctrl+N` | Новая глава |
| `Ctrl+Shift+O` | Новая книга |
| `Ctrl+O` | Открыть папку книги |
| `Ctrl+S` | Сохранить главу |
| `Ctrl+B` | Показать/скрыть сайдбар |
| `Ctrl+T` | Сменить тему (Light → Dark → Paper) |
| `Ctrl+Shift+F` | Поиск |
| `Ctrl+,` | Настройки |
| `Ctrl+Shift+.` | Режим фокуса |
| `Ctrl+Shift+Y` | Режим пишущей машинки |
| `F11` | Zen-режим |
| `Ctrl+Z/Y` | Отмена/Повтор |
| `Space+перетаскивание` | Панорамирование карты |
| `Ctrl+0` | Вписать карту в экран |
| `Escape` | Закрыть панели/палитру |

## Установка

### Требования
- [Node.js](https://nodejs.org/) v18+
- [Rust](https://rustup.rs/) stable
- Windows 10+ / macOS 12+ / Linux

### Сборка из исходников
```bash
git clone https://github.com/dontneedfriends-jpg/paddyngton.git
cd paddyngton
npm install
npm run tauri build
```

Или используйте скрипт сборки одним кликом (Windows):
```bash
.\build.bat
```

### Разработка
```bash
npm run tauri dev
```

### Тестирование
```bash
npm run test          # Юнит-тесты (Vitest, 16 тестов)
npm run test:e2e      # E2E smoke-тесты (Playwright)
```

## Почему Paddyngton?

~~Word~~ — слишком много кнопок  
~~Google Docs~~ — нужен интернет (а интернет — это отвлечение)  
~~Scrivener~~ — стоит денег  
~~Notion~~ — для стартапов, не для романов

**Paddyngton** — бесплатно, работает локально, с картой персонажей, хронологией, вики, 3 темами в стиле Acrylic и без сбора данных. Остальное — ваша работа. Удачи.
