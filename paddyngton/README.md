# 📖 Paddyngton

### Кроссплатформенный редактор для написания книг

---

*"Зачем писать книги в Word, если можно страдать с Tauri + React и получить 7 тем оформления?"*

---

## Возможности

### ✍️ Редактор
- **Markdown + LaTeX** — пишите как в 90-х, но с подсветкой синтаксиса
- **WYSIWYG** — *полу*-визуальное редактирование. Жирный выглядит жирным. Курсив — курсивом. Если повезёт
- **Undo/Redo** — Ctrl+Z существует. Ctrl+Y тоже. Потому что мы уважаем ваши ошибки
- **14 кнопок форматирования** — от ~~зачёркивания~~ до `<sup>верхнего индекса</sup>`. Зачем? Никто не знает. Но они есть

### 🗂️ Контекст персонажей
- Добавляйте персонажей, места, даты, предметы
- Группируйте их как хотите (или не группируйте)
- **Hover-подсказки** — наведите на имя — увидите детали. Наконец-то можно узнать возраст главного героя не листая 300 страниц

### 🕸️ Карта взаимодействий
- Персонажи на кружочках. Связи между ними — линиями
- Группы — отдельными кластерами с подписями
- Нажмите "Connect" → выберите двух персонажей → они теперь "связаны". Романтично

### 📚 Вики
- Справочник по всем персонажам в одном месте
- Кросс-ссылки между связанными сущностями
- Наконец-то можно понять, кто такой "он" в главе 47

### 🔍 Поиск
- Ctrl+Shift+F — ищите по всем главам разом
- Находит и контекст, и текст
- Показывает количество совпадений. Не спрашивайте зачем

### ⏱️ Версионирование
- Сохраняйте "снимки" книги с одним кликом
- Word count, char count, chapter count — для тех, кто любит статистику
- Восстанавливайте предыдущие версии. На случай "а давайте удалим всё и напишем заново"

### 🎨 7 тем оформления
- **Galaxy** — фиолетовый. Как космос, но дешевле
- **Aurora** — зелёный. Как полярное сияние, но без поездки в Норвегию
- **Forest** — коричневый. Природа.
- **Obsidian** — чёрный. Для тех, кто в танке
- **Neon** — неоновый. Как вывеска бара в 80-х
- **Retro** — как старый телевизор
- **Monochrome** — оттенки серого. Для тех, кто в депрессии

### 📝 Управление главами
- Создавайте, переименовывайте (двойной клик), удаляйте
- Вкладки в хедере — как в нормальных редакторах
- "•" — индикатор изменений. Потому что Ctrl+S существует, но лень

### ⚙️ Настройки
- Выбор шрифта из 500+ системных шрифтов — "а что если взять Comic Sans для романа?"
- Размер шрифта 10-36px — от муравьиного до "я вижу эту букву с Луны"
- Номера строк — для тех, кому нужно знать, что строка 1473 всё ещё пуста
- 3 языка: English, Español, Русский — потому что мы глобальны

## Установка

```bash
npm install
npm run tauri build
```

Или скачайте `.exe` / `.msi` / NSIS-установщик из `src-tauri/target/release/bundle/`.

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| Ctrl+N | Новая книга (создаёт папку с двумя JSON-файлами, как нормальные люди) |
| Ctrl+O | Открыть папку (любую. Даже с порно. Мы не судим) |
| Ctrl+S | Сохранить (работает. Иногда) |
| Ctrl+K | Палитра команд (нажмите и почувствуйте себя хакером) |
| Ctrl+B | Показать/скрыть сайдбар |
| Ctrl+T | Сменить тему (циклически) |
| Ctrl+Shift+F | Поиск |
| Ctrl+Z / Ctrl+Y | Отмена/повтор (CodeMirror справляется лучше вас) |
| Escape | Закрыть всё (особенно полезно после Ctrl+K) |

## Структура проекта

```
paddyngton/
├── src/                    # React-код
│   ├── App.tsx             # 1500 строк. Не судите строго
│   ├── i18n.tsx            # Интернационализация (3 языка!)
│   ├── translations/       # Переводы (включая этот README, но нет)
│   └── index.css           # 750 строк CSS. Темы. Много тем.
├── src-tauri/
│   ├── src/lib.rs          # Rust. get_system_fonts() — единственная полезная функция
│   └── Cargo.toml          # Зависимости. font-kit ради списка шрифтов.
└── package.json            # 44 строки. NPM-холивары не здесь.
```

## Формат файлов

### `.book.json`
```json
{
  "title": "Название книги",
  "author": "Автор (вы)",
  "genre": "Жанр",
  "bookType": "Novel",
  "description": "Описание. Может быть длинным.",
  "createdAt": "2024-01-01T00:00:00.000Z",
  "chapters": []
}
```

### `.context.json`
```json
[
  {
    "name": "Главный герой",
    "type": "character",
    "details": { "age": "30", "profession": "пишет код" },
    "group": "Протагонисты",
    "relations": []
  }
]
```
